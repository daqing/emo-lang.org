The Emo Programming Language

A statically typed, object-oriented, functional programming language with clean syntax and super meta programming capabilities.

Think Emo as being Go + Ruby + Python.

Hello, Emo!

The `Hello, world` example in Emo:

hello.em
  • # print "Hello, Emo!" to the console
  • print("Hello, Emo!")

or by defing a function...

hello_function.em
  • # define a function to be called later
  • # you can omit the parentheses if no params required
  • def hello_emo
  • print("Hello, Emo!")
  • end
  • hello_emo()
hello_function_params.em
  • def hello_name(name: String)
  • print("Hello, {name}!")
  • end
  • def hello_emo
  • hello_name("Emo")
  • end
  • hello_emo()
hello_function_params_with_return_value.em
  • def emo_name -> String
  • "Emo"
  • end
  • def hello_name(name: String) -> None
  • print("Hello, {name}!")
  • end
  • def hello_emo
  • hello_name(emo_name())
  • end
  • hello_emo()

or by defing a class...

hello_class.em
  • class Hello
  • # you can add `()` after the function name
  • # even without params
  • def emo() -> None
  • print("Hello, Emo!")
  • end
  • end
  • Hello.new().emo()

or by defing a class with static and private methods...

hello_class_methods.em
  • class Hello
  • # constructor
  • def init
  • pass
  • end
  • # prefix method name with an underscore(_)
  • # to define a private method
  • def _name -> String
  • "Emo"
  • end
  • # prefix method name with a dot(.)
  • # to define a static method
  • def .emo
  • print("Hello, {_name()}")
  • end
  • end
  • Hello.emo()
